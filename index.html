Backup, Export & Import

This site uses a shared Cloud API on Vercel. Admin changes are published for ALL visitors. Local export/import is still available as a backup. Use Export to download a full backup JSON file, then Import on any device/browser to restore the same content. For multi-device shared editing, keep one master backup file and re-import after changes.

Tip for Vercel: After editing content, download the backup JSON and store it safely. If you open the site on a new computer, import that file to restore apartments, projects, gallery, news, and settings.
